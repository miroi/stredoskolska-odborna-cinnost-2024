disociacna energia N2
----------------------

N2(g) = 2 N(g); De= 2*E(N) - E(N2)

E(N2,PM7)=138.03124 KJ/MOL
E(N,UHF)=472.79200 KJ/MOL
E(N,RHF)=472.79200 KJ/MOL


De= 2*E(N) - E(N2)
De= 2*472.79200 - 138.03124 =  807.55276 = 807.6 kJ/mol (ucebnica  946 kJ/mol)
